# 210 Auto Detailing

A premium, professional mobile car detailing website with a modern, luxury aesthetic.

## Features

- **Premium Design**: Clean, modern layout inspired by Tesla/Apple aesthetics
- **Fixed Navigation**: Transparent navbar that becomes solid on scroll
- **Full-Screen Hero**: Dramatic background image with centered call-to-action
- **Service Cards**: Modern hover effects with gold accent icons
- **Gallery Section**: Professional grid layout with subtle zoom effects
- **Statistics Display**: Trust-building metrics and achievements
- **Contact Section**: Clean phone number display with gold accents
- **Mobile Responsive**: Fully optimized for all device sizes

## Design Philosophy

- **Clean & Minimal**: Lots of white space, no clutter
- **Typography First**: Strong hierarchy with Inter font
- **Consistent Grid**: Everything perfectly aligned
- **Subtle Animations**: Professional hover effects only
- **Trust Building**: Statistics, professional imagery, clean layout

## Color Scheme

- **Primary**: Black (#1a1a1a), White (#ffffff), Dark Gray (#666666)
- **Accent**: Gold (#D4AF37) - used consistently throughout
- **Backgrounds**: Light gray (#f8f9fa) for sections

## Technologies Used

- HTML5 with semantic structure
- CSS3 with modern features (grid, flexbox, backdrop-filter)
- JavaScript for navbar scroll effects and smooth scrolling
- Inter font from Google Fonts
- Font Awesome icons
- High-quality stock photography

## Sections

1. **Hero**: Full-screen with background image and CTA
2. **Services**: Four service cards with hover effects
3. **Gallery**: Six professional detailing images
4. **About**: Company description with statistics
5. **Contact**: Phone numbers with call-to-action
6. **Footer**: Simple copyright notice

## Mobile Optimization

- Responsive grid layouts
- Optimized typography scaling
- Touch-friendly navigation
- Fast loading with efficient CSS

## Getting Started

Open `index.html` in your web browser to view the premium detailing website.
